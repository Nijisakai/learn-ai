# 第3节 开始无人驾驶

---

>根据训练好的神经网络模型，现在我们可以实现自动驾驶

### **程序及操作**

#### 操作步骤

1.打开终端，输入  

```bash
cd ~/Desktop/learn-ai/chapter5/self_driving_car
cd computer
python3 rc_drive_nn_only.py
```

2.新建一个终端窗口，输入  

```bash
cd ~/Desktop/learn-ai/chapter5/self_driving_car
cd raspberryPi
python stream_client.py
```
